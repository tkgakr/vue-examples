# Attribute Bindings

## 学んだこと

- `v-bind` ディレクティブで要素の属性をステートにリアクティブにバインドできる
- `v-bind:` は `:` に省略できる（例: `:title="message"`）
- `:class` へのバインドでは、単純な文字列に加えてオブジェクトや配列もサポートされている
- `:style` へのバインドも、オブジェクトや配列をサポートしている
  - 例: `:style="{ color }"` のように、プロパティ名と同名の変数はショートハンドで書ける
