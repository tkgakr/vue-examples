# Vue Examples

[Vue Examples](https://ja.vuejs.org/examples/) を Composition API、SFC、TypeScript で写経する学習用プロジェクトです。

## 構成

- Vue 3
- Vite
- TypeScript
- Vue Router
- ESLint / Prettier
- Vitest / Vue Test Utils

## セットアップ

このプロジェクトでは bun を想定しています。

```sh
bun install
```

この環境に bun がない場合は npm でも実行できます。

```sh
npm install
```

## コマンド

```sh
bun run dev
bun run build
bun run check
```

npm の場合:

```sh
npm run dev
npm run build
npm run check
```

## 検証

```sh
npm run build
npm run check
```
